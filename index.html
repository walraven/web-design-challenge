Latitude vs X

A project comparing city latitude with various aspects of the weather.

Project Explanation

The purpose of this project was to analyze how weather changes as one approaches the equator. To accomplish this analysis, data was pulled from the OpenWeatherMap API to assemble a dataset for over 500 cities.

After assembling the dataset, Matplotlib was used to plot various aspects of the weather compared to latitude. Factors analyzed include: temprature, cloudiness, wind speed, and humidity. The source data for the visualizations is also provided, in addition to brief descriptions of trends observed in the visualizations.
